A simple audio player!

How to start?

1) Download this repo
2) Open play.html
3) Choose sound
4) Relax and listen music

Stack:
    
    JS
    jQuery
    HTML5 Web Audio API
    CSS
    Bootstrap

You can:

1) Add sounds from you PC
2) Delete sounds
3) Create own playlist
4) Watch on nice visualization

P.S. Project contains folder with sounds
